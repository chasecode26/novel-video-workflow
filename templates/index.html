教程

入门指南

  • 此控制台用于管理章节、配置 provider 并启动新工作流
  • 先在文件管理中准备 input/output 文件
  • 再在仪表板通过“一键出片”启动 workflow run
  • 使用章节管理查看与编辑章节/场景数据
  • 在底部实时日志控制台中监视进度

启动章节工作流

  • 先确认章节数据已存在于数据库中
  • 在仪表板点击“一键出片”启动新的 workflow run
  • 系统会按 provider 配置依次生成音频、字幕、图片和项目文件
  • 在控制台中监视执行日志

文件管理

  • 转到文件管理标签页
  • 使用顶部按钮切换到输入(input)或输出(output)目录
  • 点击文件名可预览文本文件内容
  • 点击删除按钮可删除文件或目录
  • 使用上传按钮可上传新文件

监控进度

  • 控制台显示来自工作流与 provider 的实时日志
  • 绿色文本表示信息消息
  • 亮绿色文本表示成功消息
  • 红色文本表示错误消息
