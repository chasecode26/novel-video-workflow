教程

入门指南

  • 此控制台允许您管理和执行 MCP 服务
  • 使用 MCP 工具标签页查看和执行单个工具
  • 使用上传并处理标签页上传文件夹并运行整个工作流
  • 使用文件管理标签页管理input和output目录中的文件
  • 在每个部分底部的控制台中监视进度

执行单个工具

  • 导航到 MCP 工具标签页
  • 您将看到所有可用 MCP 工具的列表
  • 点击任何工具上的"执行"按钮单独运行它
  • 在控制台中监视执行日志

处理整个文件夹

  • 转到上传并处理标签页
  • 拖放文件夹或点击选择一个文件夹
  • 点击"处理上传的文件夹"开始工作流
  • 在控制台中监视进度

文件管理

  • 转到文件管理标签页
  • 使用顶部按钮切换到输入(input)或输出(output)目录
  • 点击文件名可预览文本文件内容
  • 点击删除按钮可删除文件或目录
  • 使用上传按钮可上传新文件

监控进度

  • 控制台显示来自 MCP 工具的实时日志
  • 绿色文本表示信息消息
  • 亮绿色文本表示成功消息
  • 红色文本表示错误消息
